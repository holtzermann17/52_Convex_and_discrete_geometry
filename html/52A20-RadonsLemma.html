Radon’s lemma


Every set A⊂ℝd of d+2 or more points can be partitioned into two disjoint sets A1 and A2 such that the convex hulls of A1 and A2 intersect.

Proof.

Without loss of generality we assume that the set A consists of exactly d+2 points which we number a1,a2,…,ad+2. Denote by ai,j the j’th component of i’th vector, and write the components in a matrix as

M=[a1,1a2,1…ad+2,1a1,2a2,2…ad+2,2⋮⋮⋱⋮a1,da2,d…ad+2,d11…1].

Since M has fewer rows than columns, there is a non-zero column vector λ such that M⁢λ=0, which is equivalent to the existence of a solution to the system

0 =λ1⁢a1+λ2⁢a2+⋯+λd+2⁢ad+2 (1)
0 =λ1+λ2+⋯+λd+2

Let A1 be the set of those ai for which λi is positive, and A2 be the rest. Set s to be the sum of positive λi’s. Then by the system (1) above

1s⁢∑ai∈A1λi⁢ai=1s⁢∑ai∈A2(-λi)⁢ai

is a point of intersection of convex hulls of A1 and A2. ∎
