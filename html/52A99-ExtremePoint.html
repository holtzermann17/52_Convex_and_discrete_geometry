extreme point


Definition.

Let C be a convex subset of a vector space X. A point x∈C is called an extreme point if it is not an interior point of any line segment in C. That is x is extreme if and only if whenever x=t⁢y+(1-t)⁢z, t∈(0,1), z≠y, implies either y∉C or z∉C.

For example the set [0,1]∈ℝ is a convex set and 0 and 1 are the extreme points.
