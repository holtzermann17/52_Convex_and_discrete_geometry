proper cone


A proper cone is a cone (http://planetmath.org/Cone3) C⊂ℝn that satisfies the following:

  • •

    C is convex;

  • •

    C is closed;

  • •

    C is solid, meaning it has nonempty interior;

  • •

    C is pointed, meaning x,-x∈C⇒x=0.

A proper cone C induces a partial ordering on ℝn:

a⪯b⇔b-a∈C.

This ordering has many nice properties, such as transitivity, reflexivity, and antisymmetry.
