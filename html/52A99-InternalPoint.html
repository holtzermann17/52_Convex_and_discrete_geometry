internal point


Definition.

Let X be a vector space and S⊂X. Then x∈S is called an internal point of S if and only if the intersection of each line in X through x and S contains a small interval around x.

That is x is an internal point of S if whenever y∈X there exists an ϵ>0 such that x+t⁢y∈S for all t<ϵ.

If X is a topological vector space and x is in the interior of S, then it is an internal point, but the converse is not true in general. However if S⊂ℝn is a convex set then all internal points are interior points and vice versa.
